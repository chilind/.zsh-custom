# Oh my zsh customs
My custimizations for oh-my-zsh.

## Installation
1. Clone this repo to {your folder}.
2. In the zsh config file `~/.zshrc` set `ZSH_CUSTOM=~/{your folder}`
3. Done!

## Prerequisites
zsh and oh-my-zsh installed.

Tip: read this guide.
https://www.sitepoint.com/zsh-tips-tricks/
